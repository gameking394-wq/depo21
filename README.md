# FitZone - Modern Fitness Merkezi Web Sitesi

Spor temalı, tamamen işlevsel ve modern bir web sitesi projesidir.

## 📋 Proje Özellikleri

### Sayfalar (4+ Sayfa)
1. **Ana Sayfa (index.html)** - Hoş geldiniz, özellikleri ve promosyonlar
2. **Hakkımızda (about.html)** - FitZone hakkında, vizyonu, ekibi
3. **Hizmetler (services.html)** - Sunulan tüm hizmetler ve paketler
4. **İletişim (contact.html)** - İletişim formu ve bilgileri

### Teknik Özellikler
- ✅ Responsive tasarım (Mobil, Tablet, Desktop)
- ✅ Modern ve kullanıcı dostu UI/UX
- ✅ Smooth scroll animasyonları
- ✅ Çalışan navigasyon sistemi
- ✅ İletişim formu
- ✅ Sosyal medya bağlantıları
- ✅ Gradient arka planlar ve modern renkler
- ✅ Font Awesome ikonları

### Renkler
- **Ana Renk (Primary)**: #FF6B35 (Turuncu)
- **İkinci Renk (Secondary)**: #004E89 (Koyu Mavi)
- **Aksan (Accent)**: #1B998B (Yeşil)

## 🚀 Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts

## 🎯 Özellikleri Kullanmak

### Navigasyon
Tüm sayfalar arasında sorunsuz geçiş yapabilirsiniz. Menü öğeleri aktif sayfa için vurgulanır.

### Hizmetler ve Fiyatlar
Hizmetler sayfasında FitZone'un sunduğu tüm hizmetler ve üyelik paketleri listelenmiştir.

### İletişim Formu
İletişim sayfasındaki form üzerinden mesaj gönderebilirsiniz.

## 📱 Responsive Design
Web sitesi tüm cihazlarda optimum şekilde görüntülenmektedir:
- Desktop (1200px ve üzeri)
- Tablet (768px - 1199px)
- Mobile (480px - 767px)
